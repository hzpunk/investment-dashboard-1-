## 1. Инвестиционные риски

### 1.1. Риск потери капитала
Все инвестиции сопряжены с риском. Вы можете потерять:
- Часть вложенных средств
- Все вложенные средства
- Получить доход ниже ожидаемого

### 1.2. Рыночные риски
- **Валютный риск** — изменение обменных курсов
- **Процентный риск** — изменение ставок ЦБ
- **Инфляционный риск** — обесценивание инвестиций
- **Ликвидности** — невозможность быстрой продажи

### 1.3. Специфические риски активов

| Тип актива | Основные риски |
|------------|----------------|
| Акции | Волатильность, банкротство эмитента |
| Облигации | Дефолт эмитента, рост ставок |
| ETF | Трекинговая ошибка, закрытие фонда |
| Криптовалюта | Экстремальная волатильность, регуляторные риски |
| Форекс | Кредитное плечо, маржин-колл |

## 4. Рекомендации по управлению рисками

### 4.1. Диверсификация
- Не вкладывайте все средства в один актив
- Распределяйте между разными классами активов
- Учитывайте географическую диверсификацию

### 4.2. Разумный объём инвестиций
- Инвестируйте только свободные средства
- Не используйте заёмные средства
- Имейте резерв на непредвиденные расходы

### 4.3. Образование
- Изучайте основы инвестирования
- Понимайте продукты, в которые инвестируете
- Следите за новостями компаний и рынка

### 4.4. Профессиональные консультации
При необходимости обратитесь к:
- Лицензированным финансовым консультантам
- Брокерам с лицензией ЦБ РФ
- Налоговым консультантам
