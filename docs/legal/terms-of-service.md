# Пользовательское соглашение

## 1. Общие положения

1.1. Настоящее Пользовательское соглашение (далее — «Соглашение») регулирует отношения между Оператором сервиса InvestTrack (далее — «Администрация») и пользователем сервиса (далее — «Пользователь»).

1.2. Использование сервиса означает безоговорочное согласие с настоящим Соглашением.

1.3. Сервис предназначен только для физических лиц, достигших 18 лет.

## 2. Определения

**Сервис** — веб-приложение InvestTrack для учёта инвестиций.

**Пользователь** — физическое лицо, использующее Сервис.

**Инвестиционные данные** — информация об активах, транзакциях, портфелях пользователя.

## 3. Предмет соглашения

3.1. Администрация предоставляет Пользователю:
- Инструменты для учёта инвестиций
- Аналитику портфеля
- Визуализацию данных
- AI-консультанта (информационный характер)

3.2. **Важно:** Сервис НЕ является:
- Инвестиционной компанией
- Брокером
- Финансовым советником
- Управляющим активами

## 4. Регистрация и аккаунт

4.1. При регистрации Пользователь обязуется:
- Предоставить достоверную информацию
- Сохранить конфиденциальность пароля
- Не передавать доступ третьим лицам
- Немедленно уведомить о несанкционированном доступе

4.2. Администрация вправе:
- Отказать в регистрации без объяснения причин
- Заблокировать аккаунт при нарушении правил
- Удалить неактивные аккаунты (после 2 лет)

## 5. Функционал сервиса

### 5.1. Учёт инвестиций
- Ручной ввод данных об активах
- Импорт данных из брокерских отчётов
- Визуализация портфеля

### 5.2. AI-консультант
- Информационная функция
- **Не является инвестиционной рекомендацией**
- Результаты требуют проверки

### 5.3. Ограничения
- Сервис не осуществляет реальные торговые операции
- Не предоставляет доступ к банковским счетам
- Не гарантирует точность рыночных данных

## 6. Уведомление о рисках

### 6.1. Инвестиционные риски
Пользователь подтверждает понимание:
- Инвестиции сопряжены с риском потери капитала
- Прошлая доходность не гарантирует будущую
- Рыночные данные могут быть неточными
- AI-консультант не даёт гарантий

### 6.2. Технические риски
- Возможны сбои в работе сервиса
- Данные могут быть потеряны (рекомендуется резервное копирование)
- Сервис не несёт ответственности за ошибки в расчётах

## 7. Интеллектуальная собственность

7.1. Все права на Сервис принадлежат Администрации.

7.2. Пользователь сохраняет права на свои инвестиционные данные.

7.3. Запрещается:
- Копирование кода сервиса
- Попытки получения несанкционированного доступа
- Использование API без разрешения

## 8. Ответственность сторон

### 8.1. Администрация не несёт ответственности за:
- Убытки от инвестиционных решений
- Неточность рыночных данных
- Технические сбои
- Действия третьих лиц

### 8.2. Пользователь несёт ответственность за:
- Достоверность введённых данных
- Сохранность пароля
- Соблюдение законодательства
- Верность инвестиционных решений

## 9. Конфиденциальность

Обработка персональных данных осуществляется в соответствии с:
- Федеральным законом № 152-ФЗ «О персональных данных»
- Политикой конфиденциальности Сервиса

## 10. Изменение и расторжение

10.1. Администрация вправе изменять Соглашение без предварительного уведомления.

10.2. Пользователь вправе удалить аккаунт в любое время.

10.3. При нарушении условий Администрация может заблокировать доступ.

## 11. Разрешение споров

11.1. Стороны примут все меры к разрешению споров путём переговоров.

11.2. При недостижении согласия — в судебном порядке по месту регистрации Администрации.

## 12. Применимое право
